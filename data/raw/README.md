# Raw Data

This directory contains original, immutable data generated by experiments.

## Contents

### conversations_json/
Individual conversation records in JSON format, each containing:
- Conversation metadata (seed prompt, category, timestamps)
- Agent configurations (models, temperatures)
- Full message history with turn-by-turn content

These are the primary raw outputs from the conversation engine.

### research_results/
Batch experiment results and metadata:
- `full_factorial_20251008_191343_results.json`: 4×6 factorial experiment (120 runs)
- `identity_archaeology_20251008_020741_results.json`: Identity archaeology template runs
- `pilot_*.json`: Early pilot test results
- `plots/`: Visualization outputs (if generated)

## Access

A symlink exists at the repository root:
- `conversations_json` → `data/raw/conversations_json`
- `research_results` → `data/raw/research_results`

## Notes

- Files are append-only; never modify raw data
- Use the analysis scripts in `analysis/` or `evaluate_conversations_json.py` to process
- Database `storage/conversations.db` contains the same data in SQLite format
